# Replay-N - Chrome Extension

用于简单快速测试重放各种HTTP请求，接口测试。

## 功能特点

- ✅ 支持多种HTTP方法：GET、POST、PUT、DELETE、PATCH
- ✅ 可添加自定义参数（支持URL参数和表单参数）
- ✅ 可添加自定义请求头
- ✅ 快速重放上次请求

## 安装方法

1. 打开Chrome浏览器，进入扩展管理页面：`chrome://extensions/`
2. 开启右上角的"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择插件所在的文件夹
5. 插件安装成功后，会在浏览器右上角显示图标

## 使用方法

1. 点击浏览器右上角的插件图标，打开插件面板
2. 在URL输入框中输入请求地址
3. 选择请求方法（GET、POST等）
4. （可选）添加请求参数：
   - 点击"Add"按钮添加参数行
   - 输入参数名和参数值
   - 不需要的参数可以点击"Remove"删除
5. （可选）添加请求头：
   - 点击"Add"按钮添加请求头行
   - 输入请求头名称和值
6. （可选）在请求体区域输入请求体内容（适用于POST、PUT、PATCH等方法）
7. 点击"Send Request"发送请求
8. 在响应区域查看响应结果，包括状态码、响应时间和响应内容
9. 点击"Replay Last"可以快速重放上次的请求
10. 点击"Clear All"可以清除所有输入内容
